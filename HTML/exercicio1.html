Exercicio 1 - textos em HTML

Esta é a lista de atividades para entendera utilização das tags na estruturação de um texto.


strong

A tag strong é utilizada para marcar conteudo com forte importancia no texto.

em

A tag em é utilizada para dar enfâse em algum conteudo importante no texto.

blockquote

A tag é utilizado para citações.

Se você quiser, se você se esforçar, se você treinar, se você entrar de cabeça, se você se concentrar... Nada garante que você vai conseguir.
-Daniel Furlan

q

Usado para citações simples.

Como Buda disse É melhor conquistar a si mesmo do que vencer mil batalhas

abbr

É usado para abreviações como por exemplo:N.A.S.A.

cite

É usado para refêrenciar algum trabalho : The Catcher in the Rye

dfn

É usado para indicar a instância de definiçaõ de um novo termo:
A fisica quantica é: um ramo teórico da ciência que estuda todos os fenômenos que acontecem com as partículas atômicas e subatômicas

address

Está tag serve para conter informações de contato:


R. Tiradentes, 654, Vitória

ins/del/s

Ins é usado para mostrar um conteudo que foi inserido, Del para mostrar o que foi excluido e s indica algo que não é mais preciso: O carro foi vendido comprado pela senhora pessoa.

sup

Este elemento é usado para conter caracteres que devem ser sobrescristos: E=MC2 é uma formula importante.

sub

Este elemento é usado para conter caracteres que devem ser subescritos: H2O é a fórmula quimica da água.
